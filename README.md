# ColaHeat

Display general sentiment of social media activity in the Columbia area in the form of a heat map.

# Tech Stack
- Database
  - MongoDB

- Data parser
  - Python 2.7
  - Tweepy - python client for twitter api
  - Textblob - python library for processing textual data

- Map Display
  - Expressjs
  - Javascript
  - HTML
  - Bootstrap
