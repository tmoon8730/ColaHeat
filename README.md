# ColaHeat

Display general sentiment of social media activity in the Columbia area in the form of a heat map.

# Tech Stack
- Database
  - MongoDB
  
- Data parser
  - Python 2.7
  - Tweepy
  - Textblob

- Map Display
  - Expressjs
  - Javascript
  - HTML
  - Boostrap
